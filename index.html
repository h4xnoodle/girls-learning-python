Functions

A function is an action that takes input and gives you output.

Sometimes input and output could be nothing!

Using a function in Python looks like this:

my_cool_function(INPUT)

The OUTPUT is sometimes nothing, a number, a word, or an action on something.

For the turtles, what kind of actions did you see?

In Python, we can use functions in two ways:

Making the 'action' happen on something (eg. a cute turtle):

turtle.forward();

Take input and give ouput:

times_two(2)

>>> 4

What is the input and output here?

Input is 2, output is 4

It takes the INPUT (the number two) and multiplies it by two. The OUTPUT is 4.

Make a function

def times_two(number):
  return number * 2

or

def lunch_time(bread, filling):
  return "For lunch I'm having a " + bread + " bread sandwich and awesome " + filling + " in it"

'return' means to give back the output

Decision Making

The key part to many things in life and in programming are the decisions we make.

With computers, there is a condition and there are different things we do, depending on what's being asked

'if statements' or 'conditional statements'

if you are finished dinner:
  you can have dessert
else:
  you need to finish your dinner first

if you have taken off your shoes:
  you can enter the house
elif you are wearing one shoe:
  take off the other shoe!
else:
  please take off your shoes!

The condition would be "you are finished dinner" which could be "yes" or "no", which in computer-speak is called TRUE and FALSE.

if an apple is brown:
  don't eat it!
elif an apple is purple:
  probably shouldn't eat it, it's purple.
else:
  eat it!

For a computer with Python

The variable 'apple' stores which colour it is.

if apple == "brown":
  print "Don't eat it!"
elif apple == "purple":
  print "Probably shouldn't eat it, it's purple."
else:
  print "Eat it!"

What would happen if the apple variable was:

apple = "brown"

apple = "green"

apple = "purple"

Spacing is important in Python! We use two spaces to tell the difference between "if" and the decision we made.

One equals sign (=) means to give a value to a variable. Two equals signs (==) means we are checking if the two things are the same

Loops

Loops allow you to write a small amount of code and have it run many times.

Like 'if' and 'else', a while loop uses a condition

while condition:
  do something cool

This is the while loop. Read it like this:

While the condition is TRUE, do something cool.

Can you think of other examples?

What if we know exactly how many times we want to "do something cool"?

for count in range(5):
  print count

You can read aloud like this:

For exactly 5 times, the variable 'count' is given the value of the current iteration (or time) we are running the loop. We then print the value of 'count'.

range(5) means that 'count' will get the value of 0, then 1, then 2... then 4.

What do you think the output is?

When you really want cake but you need to eat three servings of vegetables first...

times_ate_veggies = 3
while times_ate_veggies > 0:
  print "Eating my vegetables..."
  times_ate_veggies = times_ate_veggies - 1
print "I can have cake now!"

Remember the spaces!

">" means "greater than". Example: 5 > 2? asks: "Is 5 greater than 2?"
