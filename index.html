Functions

What is a function?

A function is an action that takes input and gives you output.

Sometimes input and output could be nothing.

An image here with INPUT > (box) > OUTPUT

Using a function in Python looks like this:

my_cool_function(INPUT)

The OUTPUT is then sometimes nothing, a number, a word, or an action on something.

For turtles, what kind of actions did you see?

Get a couple of volunteers to answer things like: moving, go up, go down, spin.

In Python, we can use actions in two ways:

Making the action happen on something (eg. a cute turtle):

turtle.moveUp();

What does this function do?

times_two(2)

4

What is the input and output here?

Input is 2, output is 4

It takes the INPUT (the number two) and multiplies it by two. The OUTPUT is 4.

Decision Making

The key part to many things in life and in programming are the decisions we make.

With computers, there is a condition and there are different things we do

For example:

if you are finished dinner:
  you can have dessert
else:
  you need to finish your dinner first

if you have taken off your shoes:
  you can enter the house
else:
  please take off your shoes!

The condition would be "you are finished dinner" which could be "yes" or "no", which in computer-speak is called TRUE and FALSE.

if an apple is brown:
  don't eat it!
else:
  eat it!

For a computer with Python

The variable apple stores which colour it is.

if apple == "brown":
  print "Don't eat it!"
else:
  print "Eat it!"

What would happen if

apple = "green"

apple = "brown"

Remember that one equals sign (=) means to give a value to a variable. Two equals signs (==) means we are checking if the two things are the same
